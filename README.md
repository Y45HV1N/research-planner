# Research Planner

This is a lightweight project planner designed for academics who juggle multiple papers, grants, and research projects at different stages of completion. This was vibe coded with the help of ChatGPT and Claude. It's designed specifically for my needs.

## Features

- **Priority-ordered projects** — drag and reorder ongoing projects by priority
- **Categories and subcategories** — organize by type (e.g., Manuscripts, Grants)
- **Project metadata** — collaborators, deadlines, target journals/grants, estimated difficulty, and requirements
- **Expandable notes** 
- **Status tracking** — move projects between "Ongoing", "Not Yet Begun", and "Completed"
- **Google Drive links** — link each project to its folder
- **Cloud sync** — automatic sync across all your devices via GitHub
- **Dark mode** 
- **Works on desktop, tablet, and phone**

(Something I have not yet implemented because I couldn't figure out a nice UI for it: I'd like to include "tasks" for each project within the projects in a sort of dynamic interactive approach... I'll add this in the future)

## Quick Start (if you know GitHub)

1. Fork this repository
2. Enable GitHub Pages (Settings > Pages > Deploy from branch > main > / (root))
3. Create a [fine-grained personal access token](https://github.com/settings/personal-access-tokens/new) scoped to your fork with Contents read/write
4. Open `https://yourusername.github.io/research-planner/` and paste your token


Note that this was designed for my needs specifically. It might be missing aspects relevant to you, or it might be too much for you.
